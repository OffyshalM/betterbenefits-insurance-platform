# betterbenefits-insurance-platform
BetterBenefits is a full-stack insurance management solution designed to streamline the relationship between policyholders and insurance providers. The platform allows users to browse coverage options, manage active policies, and track benefit distributions through a secure, centralized web interface.
Core Functionalities & Technical Implementation
Policy Management System: Developed a robust backend in PHP to handle complex insurance logic, including premium calculations and policy expiration tracking.

Relational Database Schema: Designed a MySQL database architecture to link users to multiple insurance products (Health, Life, Asset) while maintaining strict data normalization.

Automated Benefit Tracking: Integrated a dynamic dashboard that allows users to view their "Benefits" and "Coverage Limits" in real-time, pulling data asynchronously to ensure a fast UI.

User Enrollment Workflow: Built a multi-step registration and policy-selection funnel, ensuring all user inputs are validated before being committed to the database.
